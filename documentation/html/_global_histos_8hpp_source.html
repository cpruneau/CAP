16// ==================================================
17//
18// Histograms of "global" observables.
19//
20// Observables are computed in the acceptance defined by
21// particle filters. The filters can be set to select narrow
22// or broad ranges of kinematic variables as well as species
23// of particles.
24//
25// n: number of selected particles in an event
26// e: total energy of the selected particles in an event
27// q: total (net) electric charge of the selected particles in an event
28// b: total (net) baryon charge of the selected particles in an event
29//
30// The fill method of this class must be called at most once per event
31// otherwise weird multiple counting will happen...
32//
33// ==================================================
