Detailed Description

Message logger class used (inherited) by all tasks and histogram classes and subclasses of this package. Instances of this class keep track of the state of an ongoing job (controlled by by an iterator). The following states are possible:

  • TaskUnknown The state of the job is unknown, i.e., has not been set.
  • TaskOk The job is running normally.
  • TaskEof The job has reached an end of file (EoF) on input.
  • TaskEod The job has reached an end of data (EoD) on input and will now be finalized.
  • TaskWarning The job has encountered a minor error, a warning, and will continue.
  • TaskError The job has encountered an error and will be terminated gracefully if possible.
  • TaskFatal The job has encounterd a fatal error and will be aborted immediately.

Instances of this class maintain a log level variable which determines what error level will be reported on output (cout). The following levels exist

  • Unknown The report level has not been set.
  • Debug All messages produced, including debug messages, will be produced on output.
  • Info Only informational and error messages will be produced on output.
  • Warning Only warning and more severe error message will be produced on output. Debug and informational messages will be muted.
  • Error Only fatal errors reported.
  • Fatal
