bool ParticleFilterEfficiency::accept ( const Particle &  particle)
virtual

Filter particles with a parameterized efficiency

First establish this filter would accept the given particle. If it does, apply an efficiency dependent on the kinematics of the particle. pt : accept conditionally if min_pt < pt <= max_pt OR if min_pt >= max_pt eta : accept conditionally if min_eta< eta<= max_eta OR if min_eta>= max_eta y : accept conditionally if min_y< y <= max_y OR if min_y> = max_y
