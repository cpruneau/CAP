\doxysubsection{Detailed Description}
Class encapsulates the plotting properties or attributes of a graph or histograms. This class is designed to work with other WAC graphics classes to set the plotting attributes of graphs (TGraph), histograms (TH1$\ast$), and data graphs (\mbox{\hyperlink{class_data_graph}{Data\+Graph}}) displaying data values, statistical error bars, and systematic error bars.

Lines are used to draw connections between points as well as (statistical) error bars. Markers are used to represent the data point position/value Systs are used to represent the systematic error bars or bounds.
