\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_task}{Task}} used for the single purpose of loading a particle type table. The particle type and properties are loaded from a specially formatted file. The task can optionally output a copy or reduced copy of the file to a location and file name to a user specified location.

\mbox{\hyperlink{class_configuration}{Configuration}} parameters include the input file path and name of the file containing the data to be read. If the task is meant to output a copy of the file, the file path and name of the output file should also be specified. \mbox{\hyperlink{class_configuration}{Configuration}} parameters are as follows


\begin{DoxyItemize}
\item data\+Input\+Used \mbox{[}true\mbox{]} \+: whether particle type data will be read from file
\item data\+Input\+Path \mbox{[}getenv(\char`\"{}\+CAP\+\_\+\+DATA\char`\"{}) \+: folder (file path) where to find the file to be read.
\item data\+Input\+File\+Name \mbox{[}\char`\"{}\+Particle\+Type\+Data.\+dat\char`\"{}\mbox{]} \+: name of the file containing particle type PDG data to be read in.
\item data\+Output\+Used \mbox{[}true\mbox{]} \+: whether particle type data will be written to file
\item data\+Output\+Path \mbox{[}getenv(\char`\"{}\+CAP\+\_\+\+DATA\char`\"{}) \+: folder (file path) where to write a particle type PDG data file.
\item data\+Output\+File\+Name \mbox{[}\char`\"{}\+New\+Particle\+Type\+Data.\+dat\char`\"{}\mbox{]} \+: name of the file containing on output particle type PDG data. 
\end{DoxyItemize}
