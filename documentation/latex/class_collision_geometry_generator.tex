\doxysubsection{Detailed Description}
This class implements a simple heavy-\/ion collision geometry generator. The geometry of the nuclei is randomized and can be set to follow various density profile (see class \mbox{\hyperlink{class_nucleus_generator}{Nucleus\+Generator}}). This collisions generator assumes the elementary objects (either partons or nucleons) travel on straight line trajectories parallel to the z-\/axis (the beam axis). Use the configuration file to set values required at run time.
