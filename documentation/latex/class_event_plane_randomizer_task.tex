\doxysubsection{Detailed Description}
This class implements a generic event plane randomizer. All particles composing an event are azimuthally rotated by a randome angle \$\textbackslash{}varphi\$ generated event by event in the range \$\mbox{[}0,2\textbackslash{}pi\mbox{]}\$. \mbox{\hyperlink{class_event}{Event}} filters and particles filters may be used to tailor the bahvior of this randomizer.
