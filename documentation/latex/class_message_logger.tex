\doxysubsection{Detailed Description}
Message logger class used (inherited) by all tasks and histogram classes and subclasses of this package. Instances of this class keep track of the state of an ongoing job (controlled by by an iterator). The following states are possible\+:
\begin{DoxyItemize}
\item Task\+Unknown The state of the job is unknown, i.\+e., has not been set.
\item Task\+Ok The job is running normally.
\item Task\+Eof The job has reached an end of file (EoF) on input.
\item Task\+Eod The job has reached an end of data (EoD) on input and will now be finalized.
\item Task\+Warning The job has encountered a minor error, a warning, and will continue.
\item Task\+Error The job has encountered an error and will be terminated gracefully if possible.
\item Task\+Fatal The job has encounterd a fatal error and will be aborted immediately.
\end{DoxyItemize}

Instances of this class maintain a log level variable which determines what error level will be reported on output (cout). The following levels exist
\begin{DoxyItemize}
\item Unknown The report level has not been set.
\item Debug All messages produced, including debug messages, will be produced on output.
\item Info Only informational and error messages will be produced on output.
