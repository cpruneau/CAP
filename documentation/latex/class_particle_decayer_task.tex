\doxysubsection{Detailed Description}
\mbox{\hyperlink{class_task}{Task}} handles the decay of all decayable particles in the incoming event stream. At execution time, i.\+e., for each event encountered, each particle of the input stream is examined to determine whether it should be decayed. \mbox{\hyperlink{class_particle}{Particle}} deemed decayable are passed to the decay handler (decayer) and decayed. Currently, the decayer handles two and three particle decays. Decayed particles are kept in the event stream but tagged as decayed. Child particles are inserted in the stream and tagged as \char`\"{}live\char`\"{} (undecayed). Child particles that are themselves decayable are decayed iteratively.
