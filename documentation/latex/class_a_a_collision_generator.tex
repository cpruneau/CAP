\doxysubsection{Detailed Description}
MC generator producing AA collisions as superpositions of n\+Binary nucleon-\/nucleon collisions. 

\doxysubsection*{\mbox{\hyperlink{class_a_a_collision_generator}{AACollision\+Generator}} }

\mbox{\hyperlink{class_task}{Task}} designed to produce AA collisions as superpositions of n\+Binary nucleon-\/nucleon collisions. The does not generate the nucleon-\/nucleon collisions but instead call another task class that does. This task assumes an event is already loaded with \char`\"{}interactions\char`\"{} (instance of the class \mbox{\hyperlink{class_particle}{Particle}}) that carry a type for the interactions, a momentum, and a position. Nucleon-\/nucleon collision positions can for instance be generated with the \mbox{\hyperlink{class_collision_geometry}{Collision\+Geometry}} class. \begin{DoxyAuthor}{Author}
