\doxysubsection{Detailed Description}
This task computes the cumulants and nudyn values of particle multiplicities based on moments of these multiplicities previously determined with a \mbox{\hyperlink{class_nu_dyn_analyzer}{Nu\+Dyn\+Analyzer}} task. \mbox{\hyperlink{class_event}{Event}} and particle filters, histogram names, must match those generated by the \mbox{\hyperlink{class_nu_dyn_analyzer}{Nu\+Dyn\+Analyzer}} task.
