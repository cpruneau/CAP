\doxysubsection{Detailed Description}
Calculate lowest moments up to order 4 The calculation is carried out for N distinct types (species) of particles (N distinct filters)

Let the species be\+: a, b, c, etc First order moments\+: f\+\_\+1(a), f\+\_\+1(b), f\+\_\+1(c), etc Second order moments\+: f\+\_\+2(a,a), f\+\_\+2(a,b), f\+\_\+2(a,c), ..., f\+\_\+2(b,b), f\+\_\+2(b,c), etc Third order moments\+: f\+\_\+3(a,a,a), f\+\_\+3(a,a,b), f\+\_\+3(a,a,c), f(a,a,d), f(a,b,b), etc Fourth order moments\+: f\+\_\+4(a,a,a,a), f\+\_\+(a,a,a,b), f\+\_\+4(a,a,b,b), etc
