\doxysubsection{Detailed Description}
Smears the momentum of particles based on user defined configuration parameters provided at construction of instances of this class.

This class assumes the particles detected are either charged electrons, muons, pions, kaons, or protons (or either positive or negative charge). The following (arbitrary) identifiers are used internally to select which histograms or functions are used to compute resolution parameters (bias and rms) of particles. Subclass this class to add particle types as needed.
